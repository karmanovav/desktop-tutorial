Поддерживаемые форматы: JPG, PNG

Исходное изображение

Результат трансформации

Инструкция:

  1. Нажмите кнопку "Выбрать файл" для загрузки изображения
  2. Нажмите "Выбрать точки" и укажите четыре точки на исходном изображении для определения области трансформации. Выбор точек необходимо начинать с левого верхнего угла и двигаться по часовой стрелке.
  3. Нажмите кнопку "Применить трансформацию" для просмотра результата
  4. Для сохранения полученного изображения нажмите на нем правой кнопкой мыши и выберите "Сохранить изображение как..."
